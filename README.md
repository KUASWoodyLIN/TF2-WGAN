# TF2-WGAN


## Usage

- Requirements
    - [Numpy](http://www.numpy.org/)
    - [TensorFlow >= 2.0](https://www.tensorflow.org/versions/r2.0/api_docs/python/tf)
    - [TensorFlow Datasets](https://www.tensorflow.org/datasets/)

- Training WGAN-GP
    ```bash
    python main.py
    ```

- TensorBoard
    ```bash
    tensorboard --logdir logs_wgan
    ```
