# TF2-WGAN


## Usage

- Requirements
    - [Numpy == 1.19.5](http://www.numpy.org/)
    - [TensorFlow == 2.5.0](https://www.tensorflow.org/versions/r2.0/api_docs/python/tf)
    - [TensorFlow Datasets == 4.3.0](https://www.tensorflow.org/datasets/)

- Training WGAN-GP
    ```bash
    python main.py
    ```

- TensorBoard
    ```bash
    tensorboard --logdir logs_wgan
    ```
